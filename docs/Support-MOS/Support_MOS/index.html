🌟 Support MOS

🎁 Donation - Supporter Badge​

MOS is and will be free for everyone! If you would like to support the project, please consider to send a donation.

How it works​

As a thank you, every supporter who donates at least $5 will receive a code to unlock a support banner.

Validity​

  • Lifetime validity - no expiration date
  • Support banner only - no additional features (MOS is and will always be free)

How does it work?​

  1. Send a donation through our donation platform

  2. We will send you the code for the support banner via Email, Discord, Matrix or PayPal

  3. Go to Settings -> Others -> Support MOS and enter the code for the support banner

  4. The support banner will be unlocked and visible in the expandable menu at the bottom left of your MOS WebUI

💰 Donation Links​

You can donate via:

  • GitHub Sponsors: https://github.com/sponsors/ich777
  • PayPal: https://www.paypal.com/paypalme/chips777

💝 Donation via GitHub​

If you send a donation via GitHub, please contact us on Discord or Matrix so we can send you the support banner code.

📬 Contact Options​

Discord: MOS Channel

Matrix: MOS Official

🎉 Discord Role​

If you wish, you can also get a role on the Discord server so that other members can see that you have donated. This role will not show how much you donated.

🙏 Thank you for your support!​

Your donation helps us continue developing and improving MOS.
