# 0.2.3-beta

:::tip Download
Download this release on the [GitHub Releases](https://github.com/ich777/mos-releases/releases/tag/0.2.3-beta) page.
:::

## System

- Recommended Kernel: **6.1.8.20**
- Upgrade **Docker** to 29.3.1
- Upgrade **MOS Docker Watchdog** to 0.1.1
- Upgrade **MOS Notify** to 0.1.2
- Upgrade **MOS Image SHA** to 0.1.1

## Frontend

- Add PWA support
- Add file operations (copy, move, rename) to file browser
- Add file browser tooltips
- Add Help link to MOS Hub
- Add MOS Hub category filter
- Visual improvements for Pools
- Improve button layout in file browser
- Improve Icons
- Improve first setup assistant
- Refactor code structure
- Improvements to socket.io connections

## API

- Add `readme_url` to `/mos/hub/index`
- Minor fixes and cleanup
